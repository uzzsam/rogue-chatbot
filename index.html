Rogue Chatbot API

API Endpoint

This is the backend API for the Rogue Chatbot. The API endpoint is available at:

/api/chat

Usage

Send a POST request to the API endpoint with a JSON body containing a message:

{
  "message": "Your message here"
}

Response

The API will respond with a JSON object containing the chatbot's response:

{
  "response": "The chatbot's response will appear here"
}

Integration

This API is designed to be used with the Wix frontend. See the project README for integration instructions.
